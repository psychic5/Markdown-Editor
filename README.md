# Markdown Editor

A simple and lightweight markdown editor with real-time preview.

## Demo

**[Live Demo](https://psychic5.github.io/Markdown-Editor/)**

## Features

- Real-time preview
- Dark / Light theme
- File open & save
- Export to HTML
- Keyboard shortcuts (Ctrl+B, Ctrl+I, Ctrl+S)
- Resizable split view
- No dependencies - single HTML file

## Screenshot

![Markdown Editor Screenshot](screenshot.png)

## Usage

Just open `index.html` in your browser. That's it!

Or visit the [live demo](https://psychic5.github.io/Markdown-Editor/).

## Supported Markdown

- Headings (H1-H6)
- Bold, Italic, Strikethrough
- Lists (ordered, unordered, checklist)
- Code blocks
- Blockquotes
- Tables
- Links & Images
- Horizontal rules
